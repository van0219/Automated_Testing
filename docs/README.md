# Documentation

Project documentation, screenshots, and diagrams.

## Folder Structure

- **screenshots/**: UI screenshots for documentation
  - FSM navigation examples
  - Configuration screenshots
  - Error message examples
  
- **diagrams/**: Architecture and flow diagrams
  - Interface architecture
  - Data flow diagrams
  - Process workflows

## Purpose

This folder is for **documentation assets ONLY** - screenshots and diagrams used in README files and guides.

**Use this for:**
- ✅ README illustrations
- ✅ Setup guide screenshots
- ✅ Architecture diagrams
- ✅ Training materials

**Don't use this for:**
- ❌ Test execution screenshots (use TestResults/)
- ❌ Exploratory/debugging screenshots (use Temp/)
- ❌ Test evidence (use TES-070/Generated_TES070s/)
- ❌ Playwright navigation screenshots (use Temp/)
